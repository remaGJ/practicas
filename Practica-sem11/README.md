# Mini Facturador — React (STARTER)
Replica en React lo que se construyó en Svelte. Ya trae reutilizados:
`src/lib/types.ts`, `src/lib/calculos.ts` y `public/productos.json`.

## Arrancar
    npm install
    npm run dev

Completa `src/App.tsx` siguiendo la tabla Rosetta de la guía del taller.
Reactividad: useState · derivado: useMemo · carga: useEffect · evento: onClick · lista: .map()
